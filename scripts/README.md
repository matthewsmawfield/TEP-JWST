# TEP-JWST Analysis Pipeline

Reproducible analysis pipeline for testing the Temporal Equivalence Principle (TEP) using JWST high-redshift galaxy observations.

**Pipeline size:** 163 steps | **Framework:** Python 3.10+
**Last full canonical run:** ~32m18s total (2026-03-09 15:52 UTC)

---

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Step 1: Download primary data (UNCOVER DR4 — ~500 MB)
python scripts/steps/step_001_uncover_load.py

# Step 2: Run full pipeline (~32m18s in the last complete canonical run)
python scripts/steps/run_all_steps.py
```

Results appear in `results/outputs/` (JSON) and `results/figures/` (PNG).

`python scripts/steps/run_all_steps.py` now reads `results/outputs/pipeline_summary.json` at startup, prints the last full-run estimate for the complete pipeline, and shows the last-run estimate before each step. To stamp those same estimates into the live `scripts/steps/step_*.py` header comments after a successful full run, use `python scripts/steps/sync_runtime_estimates.py`.

## TEP Model

All predictions derive from:

```
Γ_t(M_h, z) = exp[ α(z) × (2/3) × (log M_h − 12) ]
```

where `α(z) = α₀ × (1+z) / (1+z_ref)`, with:
- `α₀ = 0.58 ± 0.16` (from Cepheid calibration)
- `z_ref = 0.0`
- `log M_h` = log halo mass (from stellar-to-halo mass relation)

Implemented in `scripts/utils/tep_model.py`.

## Caveats and Known Limitations

1. **z = 6–7 anomaly**: Negative mass–dust correlation (ρ = −0.12, p = 0.02) not explained by standard TEP — may reflect transition physics.
2. **Mass circularity**: Γ_t is derived from M* (via M*→M_h relation), limiting fully independent tests.
3. **Spectroscopic validation** (step_036): Marginal result (ρ = +0.18, p = 0.047) — limited by sample size.
4. **Stub steps**: 17 steps await data releases (JADES DR5 morphology catalog, COSMOS 2025, DJA Balmer decrement catalog).
5. **GOODS-S JADES v2.0**: Large file (~700 MB); step_019 downloads it automatically on first run.
