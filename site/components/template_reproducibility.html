Pipeline Steps Overview

The analysis pipeline consists of 176 deterministic steps:

  1. Step 0XX-0YY: Data acquisition and validation — downloads raw data, verifies integrity
  2. Step 0YY-0ZZ: Core analysis — statistical computations, correlations, fits
  3. Step 0ZZ-AAA: Validation suite — bootstrap tests, null tests, sensitivity analyses
  4. Step AAA-BBB: Figure generation — all manuscript plots
  5. Step BBB-CCC: Manuscript consistency checks — automated verification of all numerical claims

Each step produces JSON outputs with full metadata, and execution logs are written to logs/ with timestamps for full traceability.

Consistency Verification

The pipeline includes automated consistency checks (e.g., step_160_manuscript_consistency_check.py) that verify every numerical claim in the manuscript against the actual computed outputs. Zero tolerance for statistical-data mismatches.
