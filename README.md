## Abstract

JWST has revealed coherent anomalies at $z > 5$: star formation efficiencies exceeding $\Lambda$CDM limits, overmassive black holes, and stellar masses surpassing dynamical masses — all concentrated in the deepest gravitational potentials. This study tests whether a single violation of the isochrony axiom can account for a systematic fraction of all three. The Temporal Equivalence Principle (TEP), a chameleon-screened scalar-tensor theory, predicts enhanced proper time in unscreened potentials. With an external local Cepheid prior $\alpha_0 = 0.58 \pm 0.16$ and zero parameters tuned to JWST data, this single mechanism simultaneously predicts four distinct anomaly classes: it accounts for $\sim 34\%$ of the Red Monster efficiency excess, resolves the benchmark literature stellar-mass-function excess at $z = 6$–$10$ and 8/9 anomalous Labbé+2023 galaxies under the z-dependent correction, reduces the cosmic SFRD excess from $11.1\times$ to $2.6\times$ $\Lambda$CDM, and provides a purely kinematic mechanism (differential temporal shear) for overmassive black hole growth in Little Red Dots, with a real-data-derived regime-level dynamical-mass comparison matching the scale of the de Graaff et al. 2024 anomaly.

The primary empirical evidence comprises two lines across three surveys ($N = 4{,}726$), together with one ancillary spatial indication and one derived regime-level comparison: (L1) a dust–$\Gamma_t$ correlation ($\rho = +0.62$, $N = 1{,}283$) with $\Delta\text{AIC} \approx -4.8$ against a mass-matched threshold; (L2 - ancillary spatial indication) a preferred JADES DR5 direct-mass morphology branch with four supportive structural proxies after mass+$z$ control ($r_{\rm half,F277W/F444W}$ partial $\rho = -0.256$, Gini $= +0.361$, $\sigma_\star = +0.624$, $N = 384$), while the resolved colour-gradient branch retains the raw mass trend $\rho(M_*, \nabla_{\rm Color}) = -0.166$ and a directionally supportive debiased sign test but remains non-decisive; (L3) mass–sSFR inversion at $z > 7$; (L4) a real-data-derived regime-level resolution of the stellar-to-dynamical mass crisis ($M_* > M_{\rm dyn}$) via comparison with published kinematic literature. The sSFR sign inversion (L3) is mathematically immune to the "mass-proxy" critique: while systematic mass-measurement errors might induce a monotonic bias, they cannot produce a sharp sign inversion at $z=7$. Furthermore, L3 carries information orthogonal to dust: in UNCOVER, partial $\rho(\Gamma_t, {\rm sSFR}|{\rm dust}) = -0.49$ ($p = 10^{-18}$).

A key model-discriminating result is the Uniformity Paradox: the anomaly is not that massive galaxies at $z > 8$ are dusty, but that low-mass galaxies are *not* — in a pattern that tracks gravitational potential depth ($\rho = +0.56$). Any standard-physics resolution that tunes a time-uniform parameter (e.g., enhanced AGB yields) cannot reproduce this mass-dependent suppression: if dust production were maximally efficient everywhere, dust should be ubiquitous or track star formation, not potential depth. Only a mechanism that selectively suppresses effective time in shallow potentials — as TEP predicts via $\Gamma_t \ll 1$ for low-mass halos — reproduces the observed gradient without additional parameters.

Fitted polynomials collapse cross-survey ($R^2 = -6.4$) while $t_{\rm eff}$ generalises without training ($\rho = 0.60$–$0.80$). Full CAMB Boltzmann integration is consistent with Planck constraints ($\sigma_8$ within $0.1\sigma$). The three independent JWST surveys each confirm L1 individually above $5\sigma$ with no clustering correction required between surveys (CEERS: $z = 6.6\sigma$; UNCOVER: $z = 10.2\sigma$; COSMOS-Web: $z = 21.4\sigma$). A Fisher combination of the three photometric L1 confirmations gives $\chi^2 = 627.5$, $p = 2.8 \times 10^{-132}$, $z = 24.4\sigma$. The dedicated UNCOVER $z > 8$ validation battery independently passes all four targeted tests, including $\rho(t_{\rm eff}, A_V \mid t_{\rm cosmic}) = +0.600$ ($p = 5.0 \times 10^{-29}$), confirming that effective time rather than raw cosmic age organizes the dust signal. The functional-form test further confirms that $t_{\rm eff}$ outperforms $t_{\rm cosmic}$ in all three surveys independently at $>5\sigma$ (Steiger $Z = 5.3$, $6.7$, $16.8$), ruling out redshift ordering as the active variable. Across the broader JWST evidence package, dependence-adjusted multi-test combinations remain strongly supportive; because those omnibus figures depend on correction choice, the manuscript treats the three-survey photometric L1 replication as the headline statistic. The corrected JWST recovery gives $\alpha_0 = 0.548 \pm 0.010$ (dust, Pearson $R^2$) and $\alpha_0 = 0.548 \pm 0.010$ (joint four-observable), both consistent with the external Cepheid prior at $0.2\sigma$. The tighter JWST concordance interval reflects internal agreement among the high-redshift observables rather than a redefinition of the local prior.

Several additional supporting results extend the empirical picture. COSMOS2025 blank-field support remains strong in the raw dust relation, reaching $\rho(\Gamma_t, E(B-V)) = +0.657$ at $z = 9$–$13$ ($N = 1{,}467$), while the fully mass+redshift-controlled partial remains positive but modest ($\rho = +0.074$, $p = 4.3 \times 10^{-3}$) and is therefore treated as auxiliary support rather than as a primary partial-correlation confirmation. The cross-survey temporal-ordering test is recovered independently in all three surveys, with $\Delta\rho_{\rm time} = +0.605$ (UNCOVER), $+0.711$ (CEERS), and $+0.862$ (COSMOS-Web). UNCOVER DR4 MegaScience is null at $z < 7$, turns positive in the $z = 7$–$8$ and $z = 8$–$9$ bins ($\rho = +0.388$ and $+0.492$), and returns to null at $z = 9$–$12$. The dedicated UNCOVER dust-prediction battery passes 4/4 targeted tests. The JADES mass-to-light proxy check gives $\rho = +0.599$ raw and partial $\rho = +0.741$ after redshift control. Explicit mass debiasing strengthens previously weak signals by $\sim 1.5\times$–$1.9\times$. Key limitations: (i) $\Gamma_t$ is a deterministic function of photometric halo mass, so breaking the mass-proxy degeneracy definitively requires IFU velocity dispersions; (ii) the live DJA-based GOODS-S and Balmer replications are retained as supplementary spectroscopic checks rather than as part of the primary evidence count.

## Key Results

- **Red Monster SFE correction:** $\sim 34\%$ central correction with uncertainty propagation from $\alpha_0$.
- **Mass-density impact:** inferred cosmic stellar-mass density at $z > 7$ decreases by $\sim 19\%$.
- **Functional-form evidence:** within-regime Steiger $Z = 2.4$ ($p = 0.016$) at $z > 8$ shows $t_{\rm eff}$ outperforms $M_*$ as a dust predictor; $\Delta$AIC $\approx -5$ against a mass-matched threshold.
- **Cross-survey generalization:** polynomial baselines overfit survey-specific structure; $t_{\rm eff}$ remains stable in leave-one-survey-out tests.
- **Evidence hierarchy:** two primary empirical lines, one ancillary spatial indication, and one derived regime-level comparison (L1–L4), with additional supporting diagnostics.
- **Ancillary and derived checks:** includes inside-out screening and the regime-level dynamical-mass comparison.

## Quick Start

### Fast-Path Execution (Recommended for First-Time Users)

```bash
# Install dependencies
pip install -r requirements.txt

# Option 1: Core Evidence Only (~5 minutes)
# Validates the live evidence hierarchy (L1-L4: two primary, one ancillary, one derived)
python scripts/run_core_evidence.py

# Option 2: Photometry Anomalies Only
# Dust, UV slopes, mass-to-light ratios
python scripts/run_photometry_anomalies.py

# Option 3: Kinematics & Dynamical Masses Only
# Velocity dispersions, M*/M_dyn resolution, spectroscopic validation
python scripts/run_kinematics.py

# Full Pipeline (All 163 steps - ~32m18s in the last complete canonical run)
python scripts/steps/run_all_steps.py

# Build the manuscript site
node site/build.js
```

Pipeline outputs are written to `results/outputs/` (JSON) and `results/figures/` (PNG). Logs are saved to `logs/`.

The canonical runner reads `results/outputs/pipeline_summary.json` from the most recent successful full run and prints last-run runtime estimates both for the full pipeline and for each individual step. The same last-run estimates can be stamped into the live `scripts/steps/step_*.py` header comments with `python scripts/steps/sync_runtime_estimates.py`.

## Repository Structure

- `site/components/`: HTML components comprising the manuscript sections.
- `scripts/steps/`: Analysis pipeline (163 individual steps).
- `scripts/utils/`: Shared utilities (TEP model, data loading, plotting).
- `scripts/run_core_evidence.py`: **Fast-path for L1-L4 evidence** (~5 min).
- `scripts/run_photometry_anomalies.py`: Photometry-focused pipeline.
- `scripts/run_kinematics.py`: Kinematics & dynamical masses.
- `data/raw/`: Original survey catalogs (FITS).
- `data/interim/`: Processed CSV extracts.
- `results/outputs/`: Pipeline JSON outputs.
- `results/figures/`: Generated figures.
