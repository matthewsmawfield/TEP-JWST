# TEP-JWST Data

This directory stores input data for the High-Redshift Galaxy analysis (Paper 12).

## Required Data

1. **JADES/CEERS Catalogs:** Public releases from MAST (e.g., Labbé et al. 2023).
2. **SED Fitting Outputs:** Age, Mass, SFR estimates.

## Structure

- `jades/`: Raw JADES photometric catalogs.
- `ceers/`: Raw CEERS photometric catalogs.
- `sed/`: Derived parameters from EAZY/PROSPECTOR.
